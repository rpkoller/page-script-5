Frequently Asked Questions

Does the "Skip To Content" button always have to visible when a web page opens?
The page script and extension have configuration options for the button to not be visible when the page loads. The bookmarklets have separate links to demonstrate the configuration options.
What happens with the bookmarklets or the extension are used on pages using the SkipTo.js?
The extension and bookmarlets automatically unload any existing copies of SkipTo.js.
Can the auto-scrolling feature be disabled?
The auto-scolling feature is disabled if the operating reduced motion setting is enabled.
The is a condiguration ooption in the extension to disable auto-scrolling.
The SkipTo.js script is the modern way to implement the WCAG 2.4.1 Bypass Blocks requirement.
