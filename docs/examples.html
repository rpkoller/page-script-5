Example Configurations

Example Description
Default Menu Button SkipTo button is visible on load
Popup Menu Button The SkipTo menu button appears when it gets focus
Fixed Menu Button The SkipTo menu button is always visible at the top of the window
Headings (h1-h6) The SkipTo menu shows all H1, h2, H3, H4, H5, H6 heading levels.
Illinois Theme The color theme of the menu uses the "illinois" theme.
ARIA Theme The color theme of the menu uses the "aria" theme.
Fonts The fonts for SkipTo menu are set to 8pt and monospace.
Colors Setting the colors for skipto menu.
Position Setting the position of the SkipTo menu button from the left side of the window.
